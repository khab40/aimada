# Archived modules

These modules are intentionally excluded from active builds and runtime imports.

- `3d-lob-ui/`: experimental canvas-based LOB battlefield.
- `google-auth/`: Google OAuth, application sessions, JWTs, and identity UI.
- `advanced-controls/`: disabled advanced scenario controls retained for reference.
- `unused-modules/`: orphan components and unmounted backend routes found during the cleanup audit.
