# Kernel Shadow Mode

Kernel shadow mode was a temporary migration stage in which Python published results while Java received identical requests for comparison. It verified all six version 1 corpus scenarios before authority moved to Java.

The shadow runner, Python reference kernel, background queue, metrics, and configuration have now been removed. Java is the sole versioned-kernel implementation. Permanent verification replays the immutable corpus directly against Java with `scripts/run_java_golden_corpus.py`.
