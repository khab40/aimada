# Articles for AI Market Abuse Detection Arena

This folder contains open/public research references used for AIMADA: agent-based market simulation, order-book spoofing-like behavior, LOB realism metrics, and synthetic market simulation.

The PDFs were downloaded from public arXiv or public proceedings pages where available. Use them for project research and cite the original sources in the README/blog.

## Core references

1. ABIDES: Towards High-Fidelity Market Simulation for AI Research — arXiv:1904.12066
2. Spoofing the Limit Order Book: An Agent-Based Model — AAMAS 2017 / IFAAMAS
3. Get Real: Realism Metrics for Robust Limit Order Book Market Simulations — arXiv:1912.04941
4. Learning to simulate realistic limit order book markets from data as a World Agent — arXiv:2210.09897
5. TRADES: Generating Realistic Market Simulations with Diffusion Models — arXiv:2502.07071
6. Neural Stochastic Agent-Based Limit Order Book Simulation — arXiv:2303.00080
7. Multi-Agent Reinforcement Learning in a Realistic Limit Order Book Market Simulation — arXiv:2006.05574
8. Simulating Financial Market via Large Language Model based Agents — arXiv:2406.19966

## Project positioning note

The Nebius challenge project should not claim to be the first agent-based market simulator or a production surveillance system. A safer and stronger positioning is:

> A visual, synthetic, serverless learning arena for order-book abuse-like pattern detection and AI-generated incident explanations.
